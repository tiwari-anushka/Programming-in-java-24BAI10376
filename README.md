# 🎓 Student Management System (Java Console Project)

## 📌 Overview
This is a console-based Student Management System built using Java.  
It allows users to perform basic CRUD operations (Create, Read, Update, Delete) on student records.

---

## 🚀 Features
- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ❌ Delete Student
- 💾 Data Persistence using File Handling
- 🔁 Auto-load data on program start

---

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)
- ArrayList (Data Structure)
- File Handling (BufferedReader, BufferedWriter)


---

## ▶️ How to Run

### Step 1: Compile
javac StudentManagement.java


### Step 2: Run
java StudentManagement
